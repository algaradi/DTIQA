# LIVE (LIVE Image Quality Assessment Database Release 2)

* **Summary**: A pioneering dataset consisting of 29 high-resolution reference images distorted using 5 different types of synthetic degradation (JPEG2000, JPEG, White Noise, Gaussian Blur, Fast Fading). Totaling 779 distorted images.
* **Link**: [LIVE IQA Database Repository](https://live.ece.utexas.edu/research/quality/subjective.htm)
* **Direct download**: <https://utexas.app.box.com/v/databaserelease2>
